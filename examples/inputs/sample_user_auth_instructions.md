# Sample Project: User Authentication System

## Overview
Build a user authentication system for a web application.

## Requirements
- Users should be able to register with email and password
- Users should be able to log in
- Passwords should be stored securely
- Add password reset functionality
- Maybe include social login?

## Features
1. User registration
2. User login
3. Password hashing
4. Session management
5. Logout functionality
6. Various security measures etc.

## Questions
- What database should we use?
- Should we implement rate limiting?
- Do we need two-factor authentication?

## Technical Notes
The system should be scalable and handle several concurrent users.
